# bellatrix-auth
Authentication and authorization service for secure identity management.

**Language:** `Java`
**Stack:** `Spring Boot, Spring Security, PostgreSQL, Redis, JWT.`

```
bellatrix-auth/
├── user
├── auth
├── role
├── permission
├── session
├── token
├── email-verification
├── password-reset
└── audit
```

**Features:**
```
Register
Login
Refresh token
Logout
JWT access/refresh
Token blacklist with Redis
Email verification
Password reset
Role-based access
Organization/team membership support
```
